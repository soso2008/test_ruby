# 命名建议

* 类名和模块名用大写字母开头的单词组合
* 常量全用大写字母，用下划线分割
* 变量和参数用小写字母开头的单词组合
* 方法名全用小写字母，用下划线分割单词；方法名后加“!”表示方法具有破坏性, 可能会改变传入的参数；方法名后加“?”表示是一个布尔方法
* 类名、模块名、变量名、参数名最好使用“名词”或者“形容词＋名词”
* 方法名最好使用“动词”或者“动词＋名词”
